# Trading Bot Pro — Railway deployment

Bot de trading automático con 80 activos (crypto, acciones, ETFs, materias primas).

## Archivos
- `main.py` — bot completo + dashboard web
- `requirements.txt` — dependencias Python
- `Procfile` — comando de arranque para Railway
- `runtime.txt` — versión de Python

## Variables de entorno (configurar en Railway)
- `API_KEY` — tu Alpaca API Key
- `API_SECRET` — tu Alpaca API Secret
